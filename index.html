Walkable Southeast Asia: A Repository of Urban Liveability Features

Exploring Walkability

In the realm of urban planning and community development, understanding the walkability of a neighborhood is crucial. In this post, we'll explore a simple Python script designed to compute walkability features using a variety of powerful libraries.

Getting Started

  • rtree: A spatial indexing library that helps speed up spatial queries.
  • osmnx: A Python library to retrieve, model, analyze, and visualize street networks from OpenStreetMap.
  • geopandas: A library for working with geospatial data, enhancing the capabilities of pandas.
  • matplotlib: Used for data visualization and creating insightful maps.
  • geopy: A tool for geocoding and dealing with geographical calculations.
  • networkx: A library for the creation, manipulation, and study of complex networks.

Project Overview

The goal of our project was to develop a set of simple scripts capable of computing walkability features for a given location. We leveraged OpenStreetMap data to extract information about the street network.
